# XMonad Config

## About

XMonad is "A small but functional ICCCM-compliant tiling window manager". See
the offical [GitHub repository](https://github.com/xmonad/xmonad) for more
information.

## Requirements

Packages: `xmonad`, `xmonad-contrib`, `xmobar`

- Launcher: `dmenu`
- Terminal: `kitty`
